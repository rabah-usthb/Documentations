\tit{TD AGNES}

\vspace{0.25cm}


\begin{prettyBox}{AGNES}{myblue}
    AGNES est un algorithme de clustering hiérarchique. Il forme des clusters deux à deux jusqu'à atteindre 
    un cluster qui contient tous les points du dataset (bottom-up), ce qui donne un dendrogramme. il prend un paramètre
    $K$ (le nombre de clusters) pour définir quels clusters utiliser et le type de liaison.
\end{prettyBox}

\vspace{0.35cm}

\begin{prettyBox}{Algo}{myblue}
    \begin{enumerate}
        \setlength{\itemsep}{0pt}
        \item Calculer la distance entre chaque paire de points deux à deux et tracer la matrice d'approximité.
        \item Prendre les deux clusters avec la distance la plus petite.
        \item Supprimer la ligne et la colonne des clusters fusionnés.
        \item On met comme nouvelle distance :
            \begin{itemize}
               \setlength{\itemsep}{0pt}
                \item Min si liaison simple.
                \item Max si liaison complète.
                \item Moyenne si liaison moyenne.
            \end{itemize}
        \item Répéter jusqu'à obtenir un cluster contenant tous les points du dataset.
    \end{enumerate}
\end{prettyBox}
